About Conway's Game of Life:

Game of Life is a cellular automaton devised by Jonathan Conway, a British Mathematician, in 1970. Full details can be found on the wiki page.

How it works:

It's a "zero-player" game, which means that the initial state is given and then plays itself out based upon a set of rules. Every spot in the grid is a cell, and is either alive or dead.

Every generation (one turn) an algorithm is run that determines whether a cell will be alive or dead in the next generation. The rules are:

  • Any live cell with fewer than two live neighbours dies, as if caused by under-population.
  • Any live cell with two or three live neighbours lives on to the next generation.
  • Any live cell with more than three live neighbours dies, as if by over-population.
  • Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

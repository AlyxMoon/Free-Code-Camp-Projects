Markdown Previewer

What's Markdown?

Markdown is a super cool lightweight markup language designed to be easily formatted to HTML and many other formats. It's used for readme files, online forums, and more! It's also the formatting style of github.

This previewer uses the open-source library Marked for converting the markdown to actual HTML.

To see all the formatting options you can do with markdown, you should check out this cheatsheet on Github.
